**Skill: "Noise Cancel" / 技能名：「降噪注意力」**

**Category**: `Architecture` | **Tags**: `attention-mechanism`, `signal-noise-separation`, `transformer-efficiency`

**Source**: [papers/2024/10/2024-10-08.md, Line 9](papers/2024/10/2024-10-08.md#L9)

**EN**: NEW SKILL UNLOCKED: Noise Cancel. Standard Transformers burn compute on irrelevant tokens. Differential Transformer computes attention as the difference between two maps — signal minus noise. **CN**: 新技能解锁：降噪注意力。标准 Transformer 在无关 token 上浪费算力。Differential Transformer 把注意力算成两个 map 的差值——信号减噪声。就像给注意力戴上了降噪耳机。放大重要的，抵消不重要的。注意力分支刚刚获得主动降噪升级。

## 2. [Addition is All You Need for Energy-efficient Language Models](https://huggingface.co/papers/2410.00907)

**`[MUTATION]`** | **ArXiv**: [2410.00907](https://arxiv.org/abs/2410.00907) | **Org**: - | **Upvotes**: 151 | **Date**: 2024-10-07

**Skill: "Add, Don't Multiply" / 技能名：「加法万岁」**

**Category**: `Efficiency` | **Tags**: `L-Mul`, `integer-addition`, `compute-reduction`, `hardware-friendly`

**Source**: [papers/2024/10/2024-10-07.md, Line 9](papers/2024/10/2024-10-07.md#L9)

**EN**: MUTATION: Add, Don't Multiply. Floating point multiplication is the most expensive op in neural nets. This paper approximates multiplications with integer additions at high precision. L-Mul: O(1) multiplication using addition. The most fundamental operation in deep learning just got a massive efficiency shortcut. The compute tree mutated — multiplication branch evolved into addition.

**CN**: 变异：加法万岁。浮点乘法是神经网络里最贵的运算。这篇论文用高精度整数加法近似乘法。L-Mul：用加法实现 O(1) 乘法。深度学习最基础的运算刚刚获得巨大效率捷径。算力树发生变异——乘法分支进化成了加法。
